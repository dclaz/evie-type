# Evie Type

A simple browser-based typing game. A word and its emoji are shown on screen and spoken aloud, letter by letter as it's typed correctly, to help with early letter recognition and typing.

## Running it

No build step required. Just open `index.html` in a browser, or serve the folder locally:

```
python3 -m http.server
```

Then visit `http://localhost:8000`.

## Files

- `index.html` — page structure
- `style.css` — styling
- `game.js` — game logic
- `words.js` — word/emoji list
- `OFL.txt` — license for the embedded fonts
